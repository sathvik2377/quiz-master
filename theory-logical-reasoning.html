Logical Reasoning Theory

Develop critical thinking and analytical skills

Verbal Reasoning

Analogies

  • Word Relationships: Identify the relationship between word pairs
  • Types: Synonyms, Antonyms, Part-to-Whole, Cause-Effect
  • Function: Tool-to-Use, Worker-to-Product
  • Degree: Intensity relationships
  • Classification: Category relationships

Classification

  • Odd One Out: Find the item that doesn't belong
  • Grouping: Identify common characteristics
  • Categories: Animals, Objects, Concepts, Numbers
  • Properties: Size, Color, Function, Origin

Series Completion

  • Letter Series: Alphabetical patterns
  • Number Series: Mathematical progressions
  • Mixed Series: Combination of letters and numbers
  • Patterns: Arithmetic, Geometric, Fibonacci

Non-Verbal Reasoning

Pattern Recognition

  • Visual Patterns: Shapes, lines, and figures
  • Rotation: Clockwise and counterclockwise movements
  • Reflection: Mirror images and symmetry
  • Progression: Sequential changes in figures
  • Combination: Multiple transformations

Figure Series

  • Missing Figure: Complete the sequence
  • Next Figure: Predict the continuation
  • Transformations: Size, position, orientation changes
  • Elements: Addition, deletion, modification

Spatial Reasoning

  • Mental Rotation: Visualize 3D objects
  • Paper Folding: Predict fold patterns
  • Cube Problems: Unfolded cube patterns
  • Hidden Figures: Find embedded shapes

Logical Deduction

Syllogisms

  • Structure: Major premise, Minor premise, Conclusion
  • Types: Categorical, Hypothetical, Disjunctive
  • Validity: Logical structure correctness
  • Soundness: Valid argument with true premises
  • Venn Diagrams: Visual representation of relationships

Logical Statements

  • If-Then Statements: Conditional reasoning
  • Contrapositive: Logically equivalent statements
  • Negation: Opposite statements
  • Conjunction: AND statements
  • Disjunction: OR statements

Truth Tables

  • AND (∧): True only when both are true
  • OR (∨): False only when both are false
  • NOT (¬): Opposite truth value
  • IMPLIES (→): False only when antecedent is true and consequent is false

Critical Reasoning

Argument Analysis

  • Premise: Supporting statements
  • Conclusion: Main claim being argued
  • Assumptions: Unstated premises
  • Evidence: Facts supporting the argument
  • Counterarguments: Opposing viewpoints

Strengthen/Weaken

  • Strengthen: Add supporting evidence
  • Weaken: Introduce contradictory evidence
  • Assumptions: Identify necessary conditions
  • Alternative Explanations: Consider other possibilities

Logical Fallacies

  • Ad Hominem: Attacking the person, not the argument
  • Straw Man: Misrepresenting opponent's position
  • False Dilemma: Presenting only two options
  • Circular Reasoning: Using conclusion as premise
  • Hasty Generalization: Drawing broad conclusions from limited data

Problem Solving

Puzzle Solving

  • Grid Puzzles: Arrangement problems
  • Logic Grids: Multiple constraint satisfaction
  • Seating Arrangements: Linear and circular arrangements
  • Blood Relations: Family relationship problems
  • Direction Sense: Navigation and orientation

Mathematical Reasoning

  • Number Patterns: Arithmetic and geometric sequences
  • Coding-Decoding: Pattern-based substitutions
  • Calendar Problems: Date and day calculations
  • Clock Problems: Angle and time relationships
  • Age Problems: Relative age calculations

Data Interpretation

  • Tables: Extracting information from data tables
  • Graphs: Bar charts, line graphs, pie charts
  • Trends: Identifying patterns in data
  • Comparisons: Relative analysis of data sets
  • Calculations: Percentages, ratios, averages

Strategy and Tips

Problem-Solving Approach

  • Read Carefully: Understand the question completely
  • Identify Type: Recognize the problem category
  • Eliminate Options: Rule out obviously wrong answers
  • Work Backwards: Start from answer choices when helpful
  • Use Diagrams: Visual aids for complex problems

Time Management

  • Quick Scan: Identify easier questions first
  • Time Allocation: Don't spend too long on one question
  • Mark and Return: Come back to difficult questions
  • Educated Guessing: Eliminate options and guess
  • Practice: Regular practice improves speed

Common Mistakes

  • Misreading: Not reading questions carefully
  • Assumptions: Making unwarranted assumptions
  • Overthinking: Complicating simple problems
  • Pattern Fixation: Forcing patterns that don't exist
  • Time Pressure: Rushing through questions
