Mathematics Theory

Master mathematical concepts for BITSAT success

Algebra

Quadratic Equations

For a quadratic equation ax² + bx + c = 0:

  • Discriminant: Δ = b² - 4ac
  • Roots: x = (-b ± √Δ) / 2a
  • Sum of roots: α + β = -b/a
  • Product of roots: αβ = c/a

Sequences and Series

Arithmetic Progression (AP):

  • nth term: aₙ = a + (n-1)d
  • Sum of n terms: Sₙ = n/2[2a + (n-1)d]

Geometric Progression (GP):

  • nth term: aₙ = ar^(n-1)
  • Sum of n terms: Sₙ = a(r^n - 1)/(r - 1)
  • Sum to infinity: S∞ = a/(1-r) when |r| < 1

Trigonometry

Basic Trigonometric Ratios

  • sin²θ + cos²θ = 1
  • 1 + tan²θ = sec²θ
  • 1 + cot²θ = cosec²θ

Compound Angles

  • sin(A ± B) = sinA cosB ± cosA sinB
  • cos(A ± B) = cosA cosB ∓ sinA sinB
  • tan(A ± B) = (tanA ± tanB)/(1 ∓ tanA tanB)

Multiple Angles

  • sin2θ = 2sinθ cosθ
  • cos2θ = cos²θ - sin²θ = 2cos²θ - 1 = 1 - 2sin²θ
  • tan2θ = 2tanθ/(1 - tan²θ)

Coordinate Geometry

Straight Lines

  • Distance formula: d = √[(x₂-x₁)² + (y₂-y₁)²]
  • Section formula: (mx₂+nx₁)/(m+n), (my₂+ny₁)/(m+n)
  • Slope: m = (y₂-y₁)/(x₂-x₁)
  • Equation: y - y₁ = m(x - x₁)

Circles

  • Standard form: (x-h)² + (y-k)² = r²
  • General form: x² + y² + 2gx + 2fy + c = 0
  • Center: (-g, -f)
  • Radius: √(g² + f² - c)

Calculus

Differentiation

  • d/dx(xⁿ) = nxⁿ⁻¹
  • d/dx(sin x) = cos x
  • d/dx(cos x) = -sin x
  • d/dx(tan x) = sec²x
  • d/dx(eˣ) = eˣ
  • d/dx(ln x) = 1/x

Integration

  • ∫xⁿ dx = xⁿ⁺¹/(n+1) + C
  • ∫sin x dx = -cos x + C
  • ∫cos x dx = sin x + C
  • ∫sec²x dx = tan x + C
  • ∫eˣ dx = eˣ + C
  • ∫(1/x) dx = ln|x| + C

Probability and Statistics

Probability

  • Basic: P(E) = Number of favorable outcomes / Total outcomes
  • Addition: P(A ∪ B) = P(A) + P(B) - P(A ∩ B)
  • Multiplication: P(A ∩ B) = P(A) × P(B|A)
  • Conditional: P(A|B) = P(A ∩ B) / P(B)

Permutations and Combinations

  • Permutations: ⁿPᵣ = n!/(n-r)!
  • Combinations: ⁿCᵣ = n!/[r!(n-r)!]
  • Circular permutations: (n-1)!

Statistics

  • Mean: x̄ = Σx/n
  • Median: Middle value when arranged in order
  • Mode: Most frequently occurring value
  • Standard Deviation: σ = √[Σ(x-x̄)²/n]

Matrices and Determinants

Matrix Operations

  • Addition: (A + B)ᵢⱼ = Aᵢⱼ + Bᵢⱼ
  • Multiplication: (AB)ᵢⱼ = Σ AᵢₖBₖⱼ
  • Transpose: (Aᵀ)ᵢⱼ = Aⱼᵢ
  • Inverse: A⁻¹ = (1/|A|) × adj(A)

Determinants

  • 2×2: |A| = ad - bc
  • 3×3: Expansion by cofactors
  • Properties: |AB| = |A||B|, |Aᵀ| = |A|
